# Fifa18 CSV File to SQLite Database

 This project converts a FIFA 18 data set to an SQLite database with multiple related tables using python. The file build_fifa18_db.py creates the tables, reads the data from the fifa18.csv file, and then inserts that data into the appropriate table. 
 
 
 FIFA 18 data set used: (https://www.kaggle.com/thec03u5/fifa-18-demo-player-dataset).
